# Charis Psaltis Research Tools

This folder contains only the public GitHub Pages website files for the research
tools dashboard.

Included files:

- `index.html`
- `styles.css`
- `.nojekyll`
- project logos used by the page

## Publish with GitHub Pages

1. Create a new GitHub repository.
2. Add only the files from this folder to that repository.
2. Open the repository on GitHub.
3. Go to **Settings** > **Pages**.
4. Under **Build and deployment**, choose **Deploy from a branch**.
5. Select the branch you want to publish, usually `main` or `master`.
6. Select the root folder `/`.
7. Save.

GitHub will provide a stable public URL similar to:

```text
https://YOUR-USERNAME.github.io/YOUR-REPOSITORY/
```

That URL can be added to the UCY directory profile as the public entry point for
the modular platform.
